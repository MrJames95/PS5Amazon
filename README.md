# RdvPrefec

## Requirements
- python 3.8
- virtualenv

## How to install

Clone this project, and go inside of it.
```bash
$> virtualenv .
$> source ./bin/activate
```

and then
```bash
$> pip3 install -r requirements.txt
```

## Launch
After that you're good to go. To launch the project;
```bash
$> python3 main.py
```
